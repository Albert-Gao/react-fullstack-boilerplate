# Fullstack-react-boilerplate

> 

## Main dependencies:

This project uses:
- Feathers (As backend)
- React
- Redux
- redux-saga
- Webpack
    - JS/JSX
    - SCSS
    - Images
- Jest


## Getting Started

Getting up and running is as easy as 1, 2, 3.

1. Install the dependencies: `npm install`
2. Start the server: `npm run start`
3. Start the webpack-dev-server `npm run dev`
